Adobe Illustrator Funktsioonid

1. Pen Tool

Pen Tool on Adobe Illustratori üks mitmekülgsemaid ja võimsamaid tööriistu. Siin on mõned asjad, mida saate sellega teha.

  • Looge kohandatud kujundeid, joonistades teid ja kombineerides neid teiste kujunditega
  • Looge täpselt kõveraid ja sirgeid jooni
  • Keeruliste kujundite ja kujunduste loomiseks lisage kinnituspunkte ja manipuleerige radadega

2. Selection Tool

The selection tool võimaldab teil kujunduses objekte valida ja nendega manipuleerida. Siin on mõned selle peamised omadused:

  • Valige objektid neile klõpsates
  • Objektide teisaldamiseks, suuruse muutmiseks ja pööramiseks lohistage neid valikutööriistaga
  • Kasutage Selection Tool'iga üksikute kinnituspunktide valimiseks ja nendega manipuleerimiseks

3. Type Tool

The type tool is used for adding and formatting text in your design. Here are some of its main features:

  • Add text to your design by clicking and typing with the type tool
  • Select and format text using the character and paragraph panels
  • Use the type on a path tool to add text to a custom path or shape

4. Shape Tool

The shape tool allows you to quickly and easily create basic shapes for your design. Here are some of its main features:

  • Select from a variety of basic shapes, including rectangles, circles, and polygons
  • Resize and manipulate shapes using the selection tool
  • Combine shapes to create more complex designs

5. Pathfinder Tool

The pathfinder tool allows you to combine and manipulate shapes in more complex ways. Here are some of its main features:

  • Combine shapes using different operations, such as unite, subtract, intersect, and exclude
  • Use the shape mode and pathfinders panel to access more advanced options
  • Create custom shapes and designs by combining and manipulating shapes with the pathfinder tool
