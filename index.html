Adobe Illustrator õpetus

Kujundite joonistamine

Kujundite joonistamine on Illustratoris väga lihtne. Siin on mõned peamised funktsioonid:

  1. Tööriistaribal valige sobiv joonistusvahend, nt ristküliku või ringi joonistamise tööriist.
  2. Joonistage kujund, hoides hiireklahvi all.
  3. Vajadusel saate muuta kujundi suurust või kuju, valides selle ja kasutades transformatsioonitööriistu.
  4. Samuti saate kujundi täita erinevate värvide ja tekstuuridega, valides selle ja kasutades täitevärvi tööriistu.
  5. Lõpuks saate kujundi salvestada sobivasse failiformaati, nagu näiteks .ai või .pdf.

Tekstikujundus

Tekstikujundus on oluline osa paljudest kujundustest. Siin on mõned peamised funktsioonid:

  1. Valige tekstivahend tööriistaribalt ja looge tekstikast.
  2. Sisestage soovitud tekst ja muutke selle stiili, valides teksti ja kasutades stiilitööriistu.
  3. Muutke tekstikasti suurust ja asukohta, valides selle ja kasutades transformatsioonitööriistu.
  4. Lisage tekstile täitevärve või varje, valides selle, vali soovitud kuju ning joonista hiirega ekraanil.
  5. Kasuta kõverate joonte loomiseks kõvera vahendit - Vali tööriistade paletilt kõvera vahend ja joonista hiirega kõverad jooned.
  6. Vali ristkülikuvahend, kui soovid joonistada ristkülikuid - Vali tööriistade paletilt ristkülikuvahend ja joonista hiirega ekraanil.
  7. Kasuta ovaalivahendit, kui soovid joonistada ovaale - Vali tööriistade paletilt ovaalivahend ja joonista hiirega ekraanil.

2. Kuidas kasutada kihte

  • Lisa uus kiht - Vali kihtide paneelilt "Lisa uus kiht" nupp ning sisesta kihi nimi.
  • Nimeta kiht - Topeltkliki kihi nimel ja muuda nimi vastavalt soovile.
  • Liiguta kihti - Tõsta kiht üles või alla, et muuta selle paigutust kihtide hierarhias. Selleks vajuta kihi nimele ja lohista see sinna, kuhu soovid.
  • Peida kiht - Kui soovid, et kiht ei oleks ajutiselt nähtav, vajuta kihi silmnupule.
  • Kustuta kiht - Kui soovid kihi kustutada, tõsta kiht prügikasti ikoonile.

3. Kuidas kasutada teksti

  • Lisa tekst - Vali tekstivahend, vajuta dokumendile ja sisesta tekst. Teksti saab hiljem muuta.
  • Muuda teksti suurust - Vali tekst ja muuda suurust tööriistaribal asuvate nuppudega.
  • Muuda teksti fonti - Vali tekst ja muuda fonti tööriistaribal asuvate nuppudega.
  • Liiguta teksti - Vali tekst ja lohista see soovitud asukohta.
  • Peida tekst - Kui soovid teksti ajutiselt peita, vajuta silmnupule teksti kõrval.

4. Kuidas kasutada värvipaletti

  • Vali värv - Kui soovid värvida objekti, vali värvipaletilt sobiv värv ning vajuta objektile.
  • FAQ
    • Q: How do I create a new document in Illustrator?
    • A: You can create a new document by going to File > New or by using the keyboard shortcut Ctrl/Cmd+N.
    • Q: Can I import images into Illustrator?
    • A: Yes, you can import images by going to File > Place or by using the keyboard shortcut Ctrl/Cmd+Shift+P.
    • Q: How do I export my artwork in Illustrator?
    • A: You can export your artwork by going to File > Export or by using the keyboard shortcut Ctrl/Cmd+Shift+E.
